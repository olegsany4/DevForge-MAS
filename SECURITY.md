# Security Policy

## Сообщение об уязвимости

Пишите на security@example.com с описанием, шагами воспроизведения и оценкой влияния.

## SLA и ожидания

- Подтверждение: до 3 рабочих дней.
- Первичная оценка: до 7 рабочих дней.
- Патч/митигации: по приоритету и сложности.

## Автопроверки

- `bandit` — fail при Medium/High.
- `pip-audit` — отчёт по CVE зависимостей.
- `gitleaks` — проверка секретов (опционально, локально/CI).
